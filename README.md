## Part 1 — Imaging Technique Inspiration
Inspiration: Andy Warhol, Marilyn Monroe (1967)
Andy Warhol’s Marilyn Monroe screen prints inspire my approach because of their bold colour blocking, repetition, and pop‑art aesthetic. I want to reinterpret the piece digitally by breaking the portrait into repeated colour‑shifted segments, echoing Warhol’s mechanical reproduction style. This technique aligns well with the assignment because it allows each team member to direct a mechanic while still contributing to a cohesive visual system. The repetition and colour variation also translate naturally into generative coding techniques.

## Part 2 — Coding Technique Exploration
Technique: Perlin Noise + Randomness for Colour Variation
To recreate Warhol’s repeated, colour‑shifted Marilyn compositions, I plan to use Perlin noise combined with random colour generation. Perlin noise will allow smooth, organic transitions between colour regions, while randomness introduces the bold, unexpected palette shifts characteristic of Warhol’s prints. This technique is ideal for generating multiple variations of the same portrait, each with unique colour distortions. The p5.js example below demonstrates how noise can drive dot patterns and colour fields.
